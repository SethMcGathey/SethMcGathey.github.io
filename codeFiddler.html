Code Fiddler

This project is a clone of the popular browser coding playground, jsFiddle. This project gave Seth experience with ajax and json which he did not have much experience with before. Seth tried multiple highlighting APIs but settled on ACE because of its astetically pleasing highlighting layouts and its easy to follow documentation. Seth implemeted a repository like system to allow users to bookmark their url so that they can access it again later or make changes and then return to a past version. These commits and forks are managed with PHP and stored in a database using SQL. You can view the live project here or view the code behind this project here.

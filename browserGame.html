Browser Game

This game was a project completed by Seth McGathey, Becky Gahart, and Matt Cordier. Although roles were not implicetly defined, each person settled into roles that seemed natural. Matt handled most of the graphics and visual layout of the game, Becky handled the Node.js connections between the user screens and console screen, and Seth handled the overall game logic. The game is not currently live but you can see the code here.

Creating a Game

Game in Progress

Once a game is created. Players take their turn by clicking on a piece shown on their personal screen. The piece is placed on the board and any pieces touching this new piece move accordingly. Seth's main focus was on game logic. He created a recursive function to check players locations against the location of new squares and then move them accordingly. This required the organaiztion of multi-dimensional arrays and a numbering system for the eight possible locations on each square.

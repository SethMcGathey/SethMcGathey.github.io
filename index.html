Tsuro Clone

Along with Becky Gahart and Matt Cordier, Seth created a virtual clone of the board game Tsuro. It was created with Node.JS. This was the first time any of them had worked with Node.js. Seth feels that he learned a lot from this project and that if he created the game again with that new knowledge he could drastically improve the code, making it more simple and completing it much more quickly.

Wordpress Site

Along with Becky Gahart Scott Parker, and Matt Cordier, Seth created a wordpress site. This was the first group project of his time at DevCodeCamp that had more than one partner. It was also the most indepth he has ever worked with a wordpress site.
